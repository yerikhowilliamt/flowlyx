# Pull Request: [Phase 39] Cache Layer (Redis)

## 1. Branch Name
`feature/phase-39-cache-layer-redis`

## 2. Pull Request Title
`feat(platform): implement phase 39 cache layer (redis)`

## 3. Summary
This PR implements the Cache Layer module using Redis for the Flowlyx Platform. In compliance with the Flowlyx Engineering Handbook and the "Ponytail" guidelines, this implementation avoids excessive boilerplate and standard framework abstraction layers (like `@nestjs/cache-manager`) and instead directly integrates `ioredis`. This keeps the module extremely lightweight while giving full access to Redis features.

## 4. Files Changed
- `apps/api/package.json` (Installed `ioredis` dependency)
- `apps/api/src/modules/cache/cache.service.ts` (New service handling connection to Redis via `ioredis`)
- `apps/api/src/modules/cache/cache.module.ts` (Global Cache Module)
- `apps/api/src/app.module.ts` (Registered CacheModule)
