**5. Summary**
This PR introduces the `SettingsModule` enabling Super Admins to manage global application parameters, and standardizes the entire Flowlyx API response architecture by enforcing `@Serialize()` and `@ApiOkResponse()` across 23 controllers.

Key Changes:
- **Settings Module**: CRUD endpoints (`/settings`), guarded by `JwtAuthGuard` and `RolesGuard(Role.SUPER_ADMIN)`. Uses Zod for validation schemas.
- **Global Serialization**: Applied `@Serialize()` decorators uniformly across all API routes to strip sensitive data and standardise object shapes (e.g. `SuccessResponse`, `DomainSummary`, `DomainResponse`).
- **Response Models Bootstrap**: Generated missing standard DTO return models for all endpoint domains.
- **Cleanup**: Cleaned up repetitive Guard & Roles decorators, hoisting them to class-level configurations.

**6. Files Changed**
