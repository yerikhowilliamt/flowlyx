# UI Design (Visual Language)

Panduan visual untuk antarmuka pengguna:
- **Tema**: Dark mode eksklusif (tidak ada light mode).
- **Background**: `bg-zinc-950` untuk body utama, `bg-zinc-900` untuk panel/card.
- **Aksen**: `orange-500` (`#F97316`) untuk primary call-to-action dan sorotan.
- **Teks**: `text-zinc-50` untuk teks utama, `text-zinc-400` untuk teks sekunder.
- **Radius**: `rounded-xl` atau `rounded-lg` untuk konsistensi lengkungan.
- **Border**: Hindari border mencolok, gunakan `border-zinc-800` atau drop-shadow halus.
