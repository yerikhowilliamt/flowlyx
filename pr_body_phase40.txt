## Pull Request Description
This PR introduces the Background Workers module as per Phase 40 of the Flowlyx platform. It implements BullMQ to manage asynchronous background tasks without blocking the main thread.

## Summary
- Created `BackgroundWorkersModule`, `BackgroundWorkersService`, `BackgroundWorkersProcessor`, and `BackgroundWorkersController`.
- Registered `background-workers` queue using `@nestjs/bullmq`.
- Configured a sample `TEST_JOB` payload utilizing validation rules on DTO.
- Added comprehensive unit testing coverage for controller, service, and processor logic.
- Integrated the module into the main `AppModule`.
