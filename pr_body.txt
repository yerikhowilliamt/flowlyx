**5. Summary**
This PR implements the core functionality for Mentions as required by the Collaboration phase. It wires up the `TaskCommentsService` to automatically trigger `NotificationsService` and create notifications for users who are mentioned inside a task comment. The frontend will rely on the existing `project-members` endpoint to fetch mentionable users, adhering to the "simplest working solution" and preventing unnecessary boilerplate.
