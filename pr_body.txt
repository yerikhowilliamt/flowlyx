**5. Summary**
This PR implements the core functionality for generalized File Storage as required by the Collaboration phase. It introduces a Centralized File Entity in Prisma that associates uploaded files with a `Workspace`, `Project`, and the uploading `User`. A new `StorageModule` was created containing `StorageController`, `StorageService`, and `StorageRepository` to abstract the upload process via `CloudinaryService` and register the resulting metadata in the database.

**6. Files Changed**
- `packages/database/prisma/schema.prisma`: Added the `File` model and its relations to `User`, `Workspace`, and `Project`.
- `apps/api/src/app.module.ts`: Registered `StorageModule`.
- `apps/api/src/modules/storage/*`: Created DTOs, File Entity, Controller, Service, and Repository for handling unified file uploads.
