**5. Summary**
This PR implements the foundational Real-time Updates infrastructure using Socket.IO. It introduces the `RealtimeModule` and `RealtimeGateway` which handle client socket connections, JWT-based authentication, and room management (join/leave rooms). This centralized gateway approach adheres to the Clean Architecture guidelines and ensures REST API remains the primary mutator while the WebSocket acts as a one-way broadcasting channel to subscribed clients.
