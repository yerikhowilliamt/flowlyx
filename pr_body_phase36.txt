## Description
This PR implements the Queue System (BullMQ) module for Flowlyx as part of Phase 36. It lays down the foundation for asynchronous background job processing.

## Summary
- Added `bullmq` and `@nestjs/bullmq` dependencies.
- Added `REDIS_HOST`, `REDIS_PORT`, and `REDIS_PASSWORD` validation in Zod `env.validation.ts`.
- Integrated a `redis:7-alpine` container instance into `docker-compose.yml`.
- Created a global `QueueModule` to register the `BullModule.forRootAsync` configuration.
- Imported `QueueModule` into `AppModule`.
